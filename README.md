# UniReserva Backend API

Este es el repositorio del servidor de backend de **UniReserva**, desarrollado en Node.js utilizando Express.

## Requisitos
* [Node.js](https://nodejs.org/) instalado (v16 o superior recomendado)

## Configuración e Instalación

1. Instalar las dependencias:
   ```bash
   npm install
   ```

2. Ejecutar en modo desarrollo:
   ```bash
   npm run dev
   ```

3. Ejecutar en producción:
   ```bash
   npm start
   ```

El servidor por defecto correrá en: `http://localhost:5000`
